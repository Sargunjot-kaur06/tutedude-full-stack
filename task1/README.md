# 🌍 Hello World Task

## 📌 Project Overview
The "Hello World" project is the first milestone in the Tutedude Full-Stack journey. It focuses on the fundamental structure of an HTML document and the execution of basic tags.

## ✨ Key Features
- **Document Structure:** Implementation of `<html>`, `<head>`, and `<body>` tags.
- **Text Rendering:** Using the `<h1>` heading tag to display content on a web page.
- **Browser Compatibility:** A clean, minimal file designed to run in any modern web browser.
